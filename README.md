# 📊 Data Professional Survey Breakdown Dashboard

An interactive Power BI dashboard designed to analyze survey data from data professionals worldwide.  
This dashboard provides insights into salary trends, job roles, programming language preferences, demographics, career satisfaction, and work-life balance among data professionals.

---

## 📌 Project Overview

The Data Professional Survey Dashboard is a data analytics project built to understand the current trends and experiences of professionals working in the data industry.  

The dashboard helps explore important questions such as:
- Which data roles have the highest average salary?
- What programming languages are most preferred by professionals?
- How satisfied are data professionals with their salary and work-life balance?
- What is the distribution of data professionals across different countries?

This project demonstrates data cleaning, transformation, visualization, and storytelling using Power BI.

---

## 🛠️ Tech Stack

The dashboard was created using the following tools and technologies:

- 📊 **Power BI Desktop** – Used for creating interactive dashboards and data visualization.
- 📂 **Power Query** – Used for data cleaning, transformation, and preparing datasets.
- 🧠 **DAX (Data Analysis Expressions)** – Used to create calculated measures and KPIs.
- 🗄️ **Data Modeling** – Created relationships and structured data for effective analysis.
- 📁 **File Formats** – Power BI (.pbix) and dashboard screenshots (.png)

---

## 📂 Data Source

**Source:** Data Professional Survey Dataset

The dataset contains survey responses from data professionals, including information about:

- Country of professionals
- Job titles
- Salary details
- Programming language preferences
- Career difficulty level
- Happiness with salary
- Work-life balance satisfaction
- Age and demographic information

The dataset was cleaned and transformed before building the dashboard.

# 🚀 Dashboard Features & Highlights

## 🔹 Business Problem

The data industry contains multiple roles such as Data Analyst, Data Scientist, Data Engineer, and Database Developer. However, understanding salary trends, skill preferences, and professional satisfaction can be difficult from raw survey data.

This dashboard helps answer:
- Which roles earn higher salaries?
- Which programming languages are popular among professionals?
- How satisfied are professionals with their careers?
- What countries have the highest participation?

---

## 🎯 Goal of the Dashboard

The main objective of this dashboard is to:

- Analyze trends among data professionals globally.
- Compare salaries across different job roles.
- Understand technology preferences.
- Identify factors affecting career satisfaction.
- Present survey insights through interactive visualizations.
